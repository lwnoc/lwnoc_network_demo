

# Feature

FL-001: 32-bit address bus of AMBA3 AHB_Lite slave interface 

FL-002: supports 32/64/128/256-bit data bus (the data width corresponds to the SRAM data width)

FL-003: supports bursts of 4, 8, and 16-beats, undefined length bursts, and single transfers

FL-004: supports RMW operation for ECC

FL-005: only support single-port SRAM 

FL-006: supports SRAM capacity: 1KB~8MB

FL-007: only support same read/write latency for SRAM

FL-008: supports up to 16 GB/s write or read bandwidth @500MHz
