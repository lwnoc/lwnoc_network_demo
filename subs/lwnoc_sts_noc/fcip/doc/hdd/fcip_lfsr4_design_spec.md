# FCIP LFSR4 Design Specification

## 1. 概述

`fcip_lfsr4` 是一个4位线性反馈移位寄存器（LFSR），基于多项式 $x^4 + x + 1$ 生成伪随机序列，周期为15（排除全零状态）。

**所属分类：** basic  
**文件路径：** `ip/basic/fcip_lfsr4.sv`

## 2. 参数列表

无可配置参数。

## 3. 接口信号

| 信号名 | 方向 | 位宽 | 描述 |
|--------|------|------|------|
| clk | input | 1 | 时钟 |
| rst_n | input | 1 | 异步复位（低有效） |
| out | output | 4 | 伪随机输出 |

## 4. 功能描述

### 4.1 反馈多项式
- `feedback = out[3] ^ out[0]`，对应 $x^4 + x + 1$（最大长度LFSR）

### 4.2 移位操作
- 每个时钟上升沿：`out <= {out[2:0], feedback}`
- 复位值：`4'b0001`

### 4.3 序列特性
- 最大周期长度：$2^4 - 1 = 15$
- 不会进入全零锁死状态（复位值非零）

## 5. 验证关键点与测试用例

| TC | 场景 | 预期 |
|----|------|------|
| 001 | 复位后观察输出 | out=4'b0001 |
| 002 | 运行15拍 | 所有非零4位值各出现一次 |
| 003 | 第16拍 | 回到初始值4'b0001（周期性） |
| 004 | 复位中途插入 | 立即恢复到4'b0001 |
